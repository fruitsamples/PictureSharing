### PictureSharing ###

===========================================================================
DESCRIPTION:

This sample demonstrates how to use NSNetServices to advertise a service using Bonjour.  When used in conjunction with the PictureSharingBrowser sample code, PictureSharingBrowser clients will be able to discover and view pictures being shared by a PictureSharing server.

===========================================================================
BUILD REQUIREMENTS:

Xcode 3.2 or later

===========================================================================
RUNTIME REQUIREMENTS:

Mac OS X v10.6 or later

===========================================================================
PACKAGING LIST:

PicSharingController.h,m	 
NSObject subclass showing the use of NSNetServices to advertise a lightweight thumbnail sharing service on the network.
